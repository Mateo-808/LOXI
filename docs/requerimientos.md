# 📘 Requerimientos del Sistema - Loxi

**Nombre del Proyecto:** Loxi  
**Descripción:** Loxi es una aplicación web interactiva que busca reforzar la lógica matemática y computacional en estudiantes mediante juegos educativos, adaptados por niveles y edades.

---

## ✅ Requerimientos Funcionales (RF)

### 📂 Módulo de Autenticación
- **RF01.** El sistema debe permitir a los usuarios registrarse con nombre, correo y contraseña.
- **RF02.** El sistema debe permitir a los usuarios iniciar sesión con sus credenciales.
- **RF03.** El sistema debe validar las credenciales antes de conceder acceso.
- **RF04.** El sistema debe permitir recuperar la contraseña mediante correo electrónico.

### 📂 Módulo de Perfil
- **RF05.** El sistema debe permitir a los usuarios editar su información de perfil.
- **RF06.** El sistema debe mostrar el nombre del usuario en la interfaz principal.
- **RF07.** El sistema debe permitir cerrar sesión de forma segura.

### 📂 Módulo de Juegos
- **RF08.** El sistema debe ofrecer juegos de lógica matemática y computacional adaptados a distintos niveles escolares.
- **RF09.** El sistema debe mostrar una descripción previa de cada juego antes de iniciarlo.
- **RF10.** El sistema debe registrar el progreso del usuario en cada juego.
- **RF11.** El sistema debe desbloquear nuevos niveles a medida que el usuario completa desafíos previos.
- **RF12.** El sistema debe mostrar puntajes o recompensas obtenidas al finalizar un juego.

### 📂 Módulo de Recompensas y Avances
- **RF13.** El sistema debe otorgar gemas o puntos como recompensa por superar niveles.
- **RF14.** El sistema debe permitir usar estas gemas en una tienda para personalizar la experiencia (skins, aspectos, etc.).
- **RF15.** El sistema debe mostrar el historial de logros del usuario.

### 📂 Módulo de Contenido Educativo
- **RF16.** El sistema debe incluir explicaciones cortas o guías antes de ciertos juegos.
- **RF17.** El sistema debe sugerir juegos adecuados según el nivel de desempeño del usuario.

---

## ✅ Requerimientos No Funcionales (RNF)

### 💻 Rendimiento
- **RNF01.** El sistema debe responder a las acciones del usuario en menos de 2 segundos.
- **RNF02.** El sistema debe funcionar correctamente en los navegadores modernos (Chrome, Firefox, Edge).

### 🔐 Seguridad
- **RNF03.** Las contraseñas deben almacenarse cifradas en la base de datos.
- **RNF04.** El sistema debe prevenir ataques comunes como XSS, CSRF y SQL Injection.
- **RNF05.** El sistema debe tener protección para evitar que usuarios no registrados accedan a juegos.

### 📱 Usabilidad
- **RNF06.** El sistema debe ser responsivo y adaptarse a distintos tamaños de pantalla (móvil, tablet, escritorio).
- **RNF07.** El diseño debe ser atractivo, claro, colorido y amigable para niños y adolescentes.

### 🔧 Mantenibilidad
- **RNF08.** El sistema debe estar documentado para facilitar el mantenimiento y futuras actualizaciones.
- **RNF09.** El sistema debe permitir actualizaciones de juegos o contenido sin afectar el historial de usuario.

### 🌐 Accesibilidad
- **RNF10.** El sistema debe ofrecer textos legibles, íconos intuitivos y colores que no interfieran con la comprensión.
