# Loxi

Loxi es una plataforma desarrollada con Astro que permite medir, mejorar y aprobar la lógica de los estudiantes que desean ingresar a la media técnica.

## 🚀 Características

- Evaluaciones automatizadas para medir habilidades lógicas.
- Recursos interactivos para mejorar la lógica de programación.
- Sistema de aprobación basado en criterios de desempeño.
- Interfaz optimizada con Astro para una experiencia fluida.

## 📋 Requisitos Previos

Antes de instalar y ejecutar el proyecto, asegúrate de tener instalado:

- [Node.js](https://nodejs.org/) (versión recomendada: LTS)
- [Astro](https://astro.build/) (instalable vía npm)

## 🛠 Instalación y Configuración

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/loxi.git
   ```
2. Accede al directorio del proyecto:
   ```bash
   cd loxi
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
5. Abre en el navegador: `http://localhost:4321`

## 🖥 Uso

- Los estudiantes pueden registrarse y realizar pruebas de lógica.
- Se ofrecen explicaciones y materiales para mejorar las habilidades.
- Un sistema evalúa y aprueba a los estudiantes según su rendimiento.
